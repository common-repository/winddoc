== Description ==
Il plugin WindDoc permette la creazione di fatture elettroniche, ordini, fatture tradizionali e ricevute tramite il software <a href="https://www.winddoc.com/" rel="nofollow"> WindDoc</a>.
Tramite questo plugin potrai:
<ul>
<li>Sincronizzare gli ordini generati su WordPress direttamente in WindDoc e, se necessario, impegnare automaticamente il magazzino.</li>
<li>Aggiungere i dati per la fatturazione elettronica durante il checkout come codice destinatario ed email Pec</li>
<li>Creare una fattura in WindDoc direttamente da WordPress in modo manuale oppure automatico quando un ordine viene impostato su saldato.</li>
<li>Ogni fattura creata in WindDoc è già pronta per essere inviata come fattura elettronica.</li>
<li>Sincronizzare l’anagrafica del tuo cliente in WindDoc.</li>
<li>Impostazione la numerazione che preferisci per le tue fatture, per le tue ricevute ed i tuoi ordini.</li>
</ul>
Con l'ultimo aggiornamento il plugin permette la possibilità di integrare la gestione di prentazioni appuntamenti.
